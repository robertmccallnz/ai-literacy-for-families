## Status meanings

- **Canonical source** — the original English text. All translations should be checked against this.
- **Ready for use** — translation is competent and idiomatic; minor native-speaker polish welcome.
- **Draft — review recommended** — translation is accurate but produced without native-speaker review. Use with care; corrections welcome.
- **⚠️ Working draft — review required** — translation is a *starting point*. Do **not** use in a classroom or with families without first having a fluent speaker review it. Published openly so that fluent speakers can fix it.

## How to contribute a correction

1. Fork the repo.
2. Edit the language file.
3. Open a pull request describing what you changed and why. Add yourself as a co-translator at the bottom of the file.

If you can't use GitHub, open an [issue](https://github.com/robertmccallnz/ai-literacy-for-families/issues) describing the change, or email the maintainer.

## How to add a NEW language

This course belongs to everyone. If your community speaks a language not listed above, please add it.

1. Copy [`LANGUAGE_TEMPLATE.md`](LANGUAGE_TEMPLATE.md) to `ai_literacy_course.<lang-code>.md` using the [ISO 639 language code](https://en.wikipedia.org/wiki/List_of_ISO_639_language_codes).
2. Translate from the English source at [`../course/ai_literacy_course.md`](../course/ai_literacy_course.md).
3. Add your language to the table above.
4. Mark the status honestly — drafts are valuable, but they must be labelled.
5. Open a pull request.

### A note on Indigenous-language translations

If you are translating into an Indigenous language, please:

- **Lead the translation yourself or with your community.** Outside translators (including AI) cannot make decisions about how technical terms should enter your language.
- **Decide on technical-term policy.** For each term (e.g., "AI", "algorithm", "deepfake"), decide: keep in English, gloss, or create a new word — in consultation with elders, language workers, and teachers.
- **Use community-appropriate examples** in the activities. Replace generic examples (CEO, World War I) with ones relevant to your community.
- **Test with families and kids** before declaring a translation "ready."

The maintainer commits to crediting community translators clearly and never overriding community decisions about their own language.
